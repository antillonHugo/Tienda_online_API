Tienda Online de Ropa y Accesorios

En este proyecto, sea desarrollado un carro de compras interactivo utilizando tecnologías web como CSS, JavaScript y una API pública. El objetivo de este sistema es permitir a los usuarios agregar productos a su carro de compras y obtener información actualizada de precios y detalles de los productos a través de una API pública siendo accedida mediante cualquier dispositivo.
  • CSS: Sea utilizado CSS para diseñar y dar estilo a la interfaz del carro de compras. Esto incluye la disposición de elementos, colores, fuentes, y estilos visuales para mejorar la experiencia del usuario.
  • JavaScript: El carro de compras es completamente interactivo gracias a JavaScript. Hemos implementado funciones para agregar productos al carro, calcular el total de la compra, actualizar la cantidad de productos y gestionar eventos de interacción con los botones del carro.
  • API pública: Para obtener información actualizada de los productos, hemos utilizado una API pública que proporciona datos sobre los productos, incluyendo sus nombres, precios y detalles adicionales. La API nos permite mantener los datos del carro de compras actualizados y precisos. Dirección de la API
